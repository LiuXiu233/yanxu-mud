# 言域 0.2.0 公共 API

本文是 0.2.0 世界核心的言包导入索引。语义和不变式由各专题文档定义；源码中以 `公` 声明的类、协议、函数和错误常量均可从对应子模块使用。

0.2.x 是预览系列；公共 API、协议版本和存档格式到 1.0 才冻结。具体游戏不应导入 `src/` 内部路径，而应使用下表中的言包路径。

## 默认入口

```yanxu
引「包:言域」为 言域；

公 定 名称：文；
公 定 英文标识：文；
公 定 版本：文；
公 定 最低言序版本：文；
公 定 开发阶段：文；
公 法 服务器信息（）：典；
```

`0.2.0` 的 `服务器信息` 返回名称、英文标识、版本、最低言序版本、开发阶段和状态；状态为“预览”。

## 实体与组件

`实体` 模块的主要声明：

```yanxu
公 类 实体；
公 法 新实体（编号：文，原型编号：文）：实体；
公 法 序列化实体（实体值：实体）：典；
公 法 恢复实体（快照）：实体；
公 法 尝试恢复实体（快照）：典；
公 法 合并组件（基础：典，覆盖：典）：典；
公 法 从原型快照（编号：文，原型快照：典，动态组件：典，动态标签：列）：实体；
公 法 应用变更集（实体值：实体，变更集：典）：实体；
公 法 尝试应用变更集（实体值：实体，变更集：典）：典；
```

`实体` 实例提供原型、修订、组件、标签、快照、变更集和确认变更操作。公开错误常量包括 `编号非法错误`、`组件已存在错误`、`组件不存在错误`、`快照无效错误`、`变更集无效错误` 和 `修订冲突错误`。

`组件` 模块导出 `组件定义` 和 `注册表` 两个类。定义可取默认值、检查数据、建立新信封、规范旧信封、复制和转换为典；注册表可注册、替换、查询、规范与列出定义。错误常量为 `定义重复错误`、`定义不存在错误`、`定义无效错误`、`数据无效错误` 和 `迁移失败错误`。

`通用组件.建立注册表（）` 返回已注册 22 种 1 版通用组件的新注册表。

## 系统

```yanxu
公 类 系统；
公 类 注册表；
```

`系统` 支持启用、禁用、启动、停止、处理和装饰；`注册表` 支持注册、运行中原子替换、查询、启停、装饰、确定性排序与批量生命周期。错误常量为 `系统重复错误`、`系统不存在错误` 和 `生命周期错误`。此模块不包含移动、战斗等具体通用系统。

## 命令

`命令输入` 导出：

```yanxu
公 法 分词（原文：文）：列<文>；
公 法 解析（原文：文）：典；
公 法 尝试解析（原文：文）：典；
```

`命令` 导出 `命令定义`、`注册表` 和 `执行器`，以及：

```yanxu
公 法 新定义（配置：典，处理器）；
公 法 成功结果（命令编号：文，数据，消息：列）：典；
公 法 失败结果（命令编号：文，代码：文，消息：文，详情：典）：典；
```

`执行器.执行输入` 是从文字输入到结构化命令结果的主入口。模块公开解析、定义、重复、未知命令、权限、前置条件、冷却、上下文、处理器、中间件、结果、重入和递归深度错误常量。

## 事件

```yanxu
公 法 新事件（事件编号：文，类别：文，事件种类：文，世界时间：数，现实时间：数，来源：文，目标：文，载荷，命令编号：文，事务编号：文，追踪编号：文，内容包版本：文）：典；
公 法 验证事件（事件：典）：理；
公 类 总线；
```

`总线` 提供 `订阅`、`取消订阅`、`发布`、`入队`、`排空`、`重放`、`队列长度`、`历史` 和 `踪迹`。事件类别使用 `类别意图`、`类别事实` 和 `类别系统`；稳定错误为 `事件无效错误`、`递归超限错误` 和 `预算超限错误`。

## 消息与渲染

`消息` 导出 `新消息`、`验证消息`、`验证节点` 和 16 种节点构造函数：`文字`、`富文本`、`人物引用`、`物品引用`、`房间引用`、`命令链接`、`列表`、`表格`、`状态栏`、`进度条`、`场景描述`、`系统提示`、`错误`、`频道`、`战斗` 和 `自定义`。

`消息渲染` 导出：

```yanxu
公 法 终端安全文字（原文：文）：文；
公 法 纯文本（消息：典）：文；
公 法 ANSITelnet（消息：典）：文；
公 法 Web值（消息：典）：典；
公 法 WebJSON（消息：典）：文；
公 法 Web言据（消息：典）：文；
```

## 仓储与世界事务

`仓储` 导出 `状态仓储协议` 和 `仓储事务协议`，覆盖实体读写与查询、世界变量、计划任务、事务、快照和迁移记录。它还导出 `空状态`、`验证状态外形` 和 `验证快照外形`。错误常量区分参数无效、实体缺失/重复、修订冲突、事务冲突/状态、快照、迁移和测试故障。

`内存仓储` 导出：

```yanxu
公 类 内存仓储；
公 类 内存事务；
公 法 新仓储（）：内存仓储；
```

`内存仓储` 完整实现状态仓储协议，另提供 `导出状态`、`验证状态`、`设故障`、`清故障`、`调用记录`、`设置调用记录上限` 和 `清调用记录`。

`世界事务` 导出 `世界事务` 类和 `开始（状态仓储，编号）`。实例转发仓储事务的状态操作，并提供 `修改实体`、`暂存事件`、`暂存消息`、`待提交计数`、`提交`、`回滚` 和 `已提交输出`。

## 世界循环与虚拟时间

`世界循环` 导出 `默认预算（）` 和 `世界循环` 类。构造器注入时钟、调度器与七个阶段处理器；实例提供输入/命令/事务/事件入队、工作检查、下一唤醒时间、有预算轮转与队列状态。错误常量为 `轮次配置错误` 和 `阶段结果错误`。

`虚拟时钟` 导出：

```yanxu
公 类 虚拟时钟；
公 类 调度器；
```

`虚拟时钟` 可读取、推进或设置毫秒时间；`调度器` 可计划、取消、按预算取出到期任务、计算下一到期时间、快照与恢复。错误常量为 `时间无效错误`、`时间倒退错误`、`任务重复错误` 和 `任务无效错误`。

## 值与错误边界

世界状态、事件载荷、命令结果和结构化消息中只允许无环、处于复杂度预算内的普通言序值。数值必须有限；修订、序号、版本和预算使用协议指定的非负或正整数。公共错误以稳定 `YANYU_*` 代码与中文消息暴露，不应依赖运行时原始踪迹或源码路径。
