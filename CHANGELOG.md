# 变更记录

本项目遵循语义化版本。日期使用 ISO 8601。

## [未发布]

### 新增

- 可注入密码摘要适配器的账户服务，支持原子登录名索引、账户创建、失败限制、临时锁定、冻结、解冻和改密。
- 账户操作提交事实事件与结构化消息，公开账户投影和错误详情不返回密码或凭据摘要。
- `包:言域/账户` 公共导出及独立外部消费者契约。
- 角色服务原子维护账户角色索引与角色实体，支持数量和名称约束、完整初始组件装配，以及请求、保护期、确认或取消的删除流程。
- 新增 `角色` 通用组件和 `包:言域/角色` 公共导出；通用组件总数增至 23 种。
- 权限服务确定性合并角色组与显式授权，支持一次性管理员引导、授权轮换、末位管理员保护和命令上下文投影。
- `言域:系统/权限` 接入通用系统注册表，新增 `包:言域/权限` 外部契约与权限模型文档。
- 空间服务从静态内容图解析房间，以世界事务更新动态位置，并返回含可见实体与出口的结构化场景消息。
- 新增可替换的移动、观察系统及 `包:言域/空间` 外部契约；新角色默认装配可移动组件。

## [0.4.1] - 2026-07-17

### 修复

- 从干净克隆生成外部消费者路径依赖锁，避免本地忽略制品污染包摘要。
- macOS 使用 Python 标准库计算 SQLite 官方制品 SHA3-256，绕开平台不支持的 .NET 实现。
- 本地验证制品写入言序规范排除的 `.yanxu/verify`，重复验证不再使路径依赖锁失效。

## [0.4.0] - 2026-07-17

### 新增

- 内容寻址言据文件仓储与 SQLite 原子世代仓储，状态、快照和事件日志同世代提交。
- 哈希链事实事件日志，以及查询、窗口读取和确定性重放。
- 有严格历史前缀、定义摘要、后置检查、批次回滚和并发冲突保留的数据迁移执行器。
- 三后端统一的快照恢复预检与恢复后状态摘要验证报告。

### 变更

- 根包版本和服务器信息更新为 0.4.0“持久化”预览。
- 状态仓储公共协议增加只读恢复预检，恢复操作返回前后状态摘要验证报告。
- 公共导出增加数据迁移模块，外部消费者契约固定新的导出表与包摘要。

## [0.3.0] - 2026-07-17

### 新增

- 内容包清单、语义版本约束、确定性依赖排序，以及跨包可见性和引用图。
- 声明式内容 Schema、内容文档与批量文档集、定义注册表和合并后 Schema 校验。
- 多级原型继承、本地化目录、语言回退、插值、复数接口和缺失键检测。
- 带分项指纹、总摘要、统计和隔离查询的内容图，以及差异与传递影响分析。
- 含静态策略、现场影响检查、代数冲突保护、原子提交、系统事件和审计记录的热重载运行时。
- 有路径、文件、项目和总量预算的文件内容包加载器，检测包归属、语言错配和读取期间文件变化。
- 中文内容检查、内容构建、内容打包和内容差异 CLI，以及带 SHA-256 摘要和回读校验的言据制品。
- 独立青石镇示例内容，包含 37 项定义、13 个房间、多级物品原型、本地化和跨内容引用。
- 12 个内容运行时公共子模块及其导入契约测试。

### 变更

- 根包版本和服务器信息更新为 0.3.0“内容运行时”预览。
- 最低工具链更新为言序 1.1.12，使用其显式执行步数预算完成大型内容制品构建。
- 内容装载和构建复用可信规范值、索引与确定性内部指纹，降低默认预算下的重复序列化和隔离复制。

## [0.2.0] - 2026-07-16

### 新增

- 实体快照、变更集、原型合并和未知组件往返；组件注册表支持 Schema、默认值和逐版迁移。
- 22 种通用组件定义，以及可排序、禁用、替换、装饰和管理生命周期的系统注册表。
- 命令输入、定义、注册和执行协议，含权限、前置条件、冷却、中间件和结构化错误。
- 领域事件信封、同步订阅、队列、确定性重放、递归限制、事件预算和结构化踪迹。
- 含 16 种节点的结构化消息协议，以及纯文本、ANSI Telnet、Web JSON 和 Web 言据渲染器。
- 世界事务和提交后发件箱，事件驱动世界循环，以及可注入虚拟时钟与可恢复调度器。
- 状态仓储与仓储事务协议，以及支持查询、乐观修订、快照、迁移记录和故障注入的内存实现。
- 根包公开导出 14 个经过验证的世界核心子模块。

### 变更

- 根包版本与运行时版本信息更新为 0.2.0，状态标记为“世界核心预览”。
- 状态、消息、事件、命令和世界阶段输出统一拒绝非有限数、循环值和超出复杂度预算的输入。

## [0.1.0] - 2026-07-16

### 新增

- 言域格式 2 工程、公共入口与中文 CLI 版本命令。
- 精确修订的言序生态依赖清单和兼容边界。
- 跨平台 CI、发布和安全检查基线。
- 架构、入门、贡献、安全和兼容文档。
