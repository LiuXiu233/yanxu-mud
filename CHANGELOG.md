# 变更记录

本项目遵循语义化版本。日期使用 ISO 8601。

## [未发布]

### 计划

- 0.3 内容包、原型、跨包引用、本地化和热重载。

## [0.2.0] - 2026-07-16

### 新增

- 实体快照、变更集、原型合并和未知组件往返；组件注册表支持 Schema、默认值和逐版迁移。
- 22 种通用组件定义，以及可排序、禁用、替换、装饰和管理生命周期的系统注册表。
- 命令输入、定义、注册和执行协议，含权限、前置条件、冷却、中间件和结构化错误。
- 领域事件信封、同步订阅、队列、确定性重放、递归限制、事件预算和结构化踪迹。
- 含 16 种节点的结构化消息协议，以及纯文本、ANSI Telnet、Web JSON 和 Web 言据渲染器。
- 世界事务和提交后发件箱，事件驱动世界循环，以及可注入虚拟时钟与可恢复调度器。
- 状态仓储与仓储事务协议，以及支持查询、乐观修订、快照、迁移记录和故障注入的内存实现。
- 根包公开导出 14 个经过验证的世界核心子模块。

### 变更

- 根包版本与运行时版本信息更新为 0.2.0，状态标记为“世界核心预览”。
- 状态、消息、事件、命令和世界阶段输出统一拒绝非有限数、循环值和超出复杂度预算的输入。

## [0.1.0] - 2026-07-16

### 新增

- 言域格式 2 工程、公共入口与中文 CLI 版本命令。
- 精确修订的言序生态依赖清单和兼容边界。
- 跨平台 CI、发布和安全检查基线。
- 架构、入门、贡献、安全和兼容文档。
